Final Micro Interaction Explanation

Link: http://reiddumont.com/portfolio.html
Triggers: User clicks either arrow pointing left or right.
Rules: When the user clicks an arrow, the image in the carousel will change. Clicking an arrow opposite of the previous clicked will take the user to the previous image they saw. If the user goes through all the images and clicks the same arrow again, the carousel will repeat from the first image.
Feedback: The image being displayed will change.
Loops and Modes: As long as the user keeps clicking the arrow, the carousel will cycle. No modes are present.
