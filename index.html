Alpha Micro Interaction Explanation

Link: https://xivstyle.com/
Triggers: User hovers mouse cursor over drop down text.
Rules: If the user's mouse cursor leaves the area of the button, then the text will no revert to no longer being highlighted. If the button is clicked and the drop down is engaged, then the button will remain highlighted until the drop down is disengaged.
Feedback: The button will change colors to indicate a highlight.
Loops and Modes: When the drop down button is clicked more buttons are shown.
